python变量

 2018-03-29 | 开发 · 后端 · python · 

创建变量时会在内存中开辟一个空间。基于变量的数据类型，解释器会分配指定内存。

python的数据类型

python有五个标准的数据类型：

  • numbers：数字
  • string：字符串
  • list：列表
  • tuple：元组
  • dictionary：字典

numbers

list

列表用[]标识。列表是有序的对象集合。

tuple

元组用()标识，内部元素用,隔开。

dictionary

字典用{}标识。字典由索引（key）和它对应的值value组成，字典是无序的对象集合。
